Giflist is a client for reddit that will stream GIFs from any subreddit that contains gifs. By default it used the popular gifs subreddit, but you can change this to whatever you like, e.g: perfectloops, chemicalreactiongifs, animalsbeingderps.


To play a GIF, just tap it. To view the original submission for any GIF, tap the title section and it will open in reddit. You can configure some settings for the application below. Enjoy!
